== Description ==

The plugin will give you an easy access to all your typeforms. 
You can easily import your forms with giving the plugin your custom URL:

*   The plugin will automaticly generate a shortcut code
*   You can choose if you want to implement it with an iframe, drawer or with a classic pop-up
*   Paste the shortcut everywhere you want and use your custom typeform

== Installation ==

1. Upload `typeform/typeform.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Plugins -> Typeform' for using our shortcut helper

== Frequently Asked Questions ==

= Can I use my own  shortcuts? =

Absolutely, you can use your own shortcut.
It's formats is: '[typeform data="Typeform URL" text="Button text" type="typeform Type"]
Supported typeforms type:
*   classic
*   frame
*   drawer
